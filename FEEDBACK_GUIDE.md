# Hanemy β v0.9.1 フィードバック回収メモ

他の人に触ってもらうときは、「良い？」ではなく「どこで迷った？」を聞きます。  
好意的な感想より、止まった場所の方が改善に使えます。

## 聞くこと

1. 最初に何をすればいいか分かりましたか？
2. 「今月あと使えるお金」はすぐ見つかりましたか？
3. 「貯蓄」と「今月あと使えるお金」の違いは分かりましたか？
4. 今日使った分の入力は面倒でしたか？
5. 超過時の表示は分かりやすかったですか？
6. 親共有カードは自然に送れそうですか？
7. 毎日開くには重い・うるさいと感じた場所はありますか？

## 見ること

- 初回設定に何秒かかったか
- どの画面でスクロールが止まったか
- どの文言を読み飛ばしたか
- どのボタンを押すか迷ったか
- スマホ下部ナビを自然に使ったか

## 聞かなくていいこと

- 欲しい機能を自由に聞きすぎない
- デザインの好みを深追いしすぎない
- グラフや履歴が欲しいかを最初から聞かない

v0.9.1では、機能の追加要望よりも、説明なしで使えるかを優先して確認します。
