# Hanemy β v1.0.2 Fixed Presets

ハネミーは、大学生向けの「今月あといくら使えるか」を可視化する生活費ボードです。  
vv1.0.2では機能追加ではなく、ブランドキャラクターとアプリUIの統一を行いました。

## 変更点

- ホーム画面をブランドキャラクター中心の残額ボードへ再構成
- 「今月あと使えるお金」を主役カードとして整理
- 「今日使うならこのくらいまで」「貯蓄」を小カード化
- 入力ボトムシートを「今日使った分を足す」UIへ刷新
- 「今月の準備」を4ステップカード化
- 親共有画面と共有カードの見た目をブランドトーンに統一
- 下部ナビは「ホーム / 入力 / 共有」の3項目を維持
- App Store / Google Play 表記なし
