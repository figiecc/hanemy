# Hanemy β v0.9 Public Beta

ハネミーは、大学生向けの「今月あといくら使えるか」を可視化する生活費ボードです。  
家計簿のように細かく記録するアプリではなく、月末まで生活費が持つかを判断するための軽量ツールです。

## できること

- 今月あと使える生活費を確認する
- 今日使った分だけ、ざっくり追加する
- 生活費を超えた場合に、今月の貯蓄予定額への影響を見る
- 全体額まで超過した場合だけ、貯蓄から明示的に補填する
- 使わなかった分を、月初更新時に貯蓄へ回す
- 親に見せやすい生活費カードを作る

## 使い方

1. 基本テンプレートで始める
2. 今月入るお金を入力する
3. 先に消えるお金を入力する
4. 日々の支出は「入力」からざっくり足す
5. 新しい管理期間になったら、月初更新カードから今月を始める

## このアプリが家計簿ではない理由

ハネミーは、細かい支出履歴やレシート管理を目的にしていません。  
目的は「今月の生活費が月末まで持つか」をすぐ判断できることです。

そのため、履歴一覧・グラフ・詳細分析はあえて入れていません。

## データ保存について

- データはブラウザの端末内ストレージに保存されます
- ログイン、クラウド同期、外部送信はありません
- ブラウザのデータ削除や別端末への移動では、保存内容が消える可能性があります

## GitHub Pagesで公開する場合

このフォルダ内のファイルをリポジトリ直下に置けば、そのまま静的サイトとして公開できます。  
公開前に `TEST_CASES.md` と `PUBLISH_CHECKLIST.md` を確認してください。

## β版でまだ入れていないもの

- 決済
- 広告
- ログイン
- クラウド同期
- 月次レポート
- 過去月一覧
- 詳細な支出履歴
- グラフ
- テーマ変更

## v0.9 の目的

v0.9 は public beta です。  
目的は、機能を増やすことではなく、自分以外の人がスマホで開いて説明なしで使えるかを確認することです。

見るべき点は、次の3つです。

1. 初回設定で迷わないか
2. 「今月あと使えるお金」と「貯蓄」の違いが伝わるか
3. 入力と親共有がスマホで自然に使えるか
