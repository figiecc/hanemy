# Hanemy β v0.8.1 Overage Savings

大学生向けの「今月あといくら使えるか」を可視化する生活費ボードです。

## v0.8.1 の目的
- 生活費超過時の扱いを整理
- 貯蓄を勝手に減らさず、必要なときだけ明示的に補填できるようにする
- 使わなかった生活費と今月の貯蓄分を、月初更新時に累積貯蓄へ回す
- 金額ロジックの信頼性を上げる

## v0.8.1 の主な変更
- 生活費枠を超えた場合、今月貯蓄に回る予定額が自動で減るように整理
- 収入−固定費の全体額まで超えた場合だけ、「貯蓄から補填する」ボタンを表示
- 貯蓄補填は自動ではなく、ユーザーが押したときだけ実行
- 月初更新時の貯蓄追加額を「収入−固定費−使った額」の残りに修正
- 補填操作の直前取り消しに対応
- キャッシュ更新用に `v=081` を付与
