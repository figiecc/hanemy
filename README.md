# Hanemy β v0.7.5 Final Consistency

貯蓄ロジックを整理し、UI上の重複導線を削った最終整合版です。

## 主な変更
- 生活費とは別に「貯蓄」を先に確保
- 「今月あと使えるお金」は、貯蓄分を除いた生活費ベースで表示
- 貯蓄は月初更新時に「今月分の貯蓄配分 + 使わずに残った生活費」を累積
- ホームに貯蓄の累積額を表示
- 上部の準備ジャンプボタンを削除
- スマホ下部ナビの設定ボタンを削除
- 月初更新後の完了文言を貯蓄ロジックに合わせて調整
- キャッシュ更新用に v=075 を付与

## 基本テンプレート配分
- 食費35%、交通費10%、交際費10%、デート代5%、趣味7%、服・美容5%、勉強3%、貯蓄25%
