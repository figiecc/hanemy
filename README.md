# ハネミー β v0.15 release beta candidate

## この版の目的

機能追加ではなく、リリースβ候補として機能面を一旦固定するための版です。

## 主な変更

- サンプルデータなし
- debug表記なし
- 初回ホーム導線を公開前提に調整
- フィードバック先メールを `FEEDBACK_EMAIL` 定数で差し替えやすく変更
- フィードバック本文にバージョンを自動挿入
- データ削除・リセット確認文言を公開前提に調整
- ローカル保存型である注意を設定内に維持
- お知らせに v0.15 の内容を追加

## リリースβの機能完成条件

- 初回ユーザーが今月の準備を完了できる
- ホームで今月あと使えるお金が分かる
- ざっくり入力で使った金額を反映できる
- 間違えた入力を取り消せる
- 今月やりたいことを編集できる
- ふり返りから次の期間へ引き継げる
- データはローカル保存される
- フィードバックを送れる

## 公開前に必ず差し替えるもの

`index.html` 内の以下を運営用メールアドレスに変更してください。

```js
const FEEDBACK_EMAIL='example@example.com';
```

## 公開前チェック

1. 初回ホームから「今月の準備」を開始できる
2. 今月の準備を完了するとホームに戻る
3. ホームに「今月あと使えるお金」が表示される
4. ざっくり入力で金額を追加できる
5. ざっくり入力で直後の取り消しができる
6. 履歴から削除できる
7. 今月やりたいことを編集できる
8. ふり返りから次の期間を作れる
9. 設定から期間変更・リセット・全削除ができる
10. フィードバックを送れる
11. ブラウザを再読み込みしてもデータが残る

## v0.16 PWA stable candidate

この版は、ホーム画面追加によるアプリ感を維持しつつ、更新詰まりを減らすためのPWA安定版です。

### 方針

- `manifest.json` とアイコンは残す
- `sw.js` は使う
- ただし `index.html` はキャッシュしない
- キャッシュするのは画像・アイコンなどの安定素材だけ
- 古いキャッシュはService Workerのactivate時に削除する

### できること

- スマホのホーム画面に追加できる
- アプリ風に起動できる
- アイコンやロゴは安定して表示されやすい
- 本体HTMLは最新版を取りに行きやすい

### できないこと

- 完全オフライン起動
- App Storeアプリと同等の強制更新保証

## バージョン

Hanemy β / v0.16 PWA stable candidate


## v0.16.1 settings fix

設定画面のボタンが動かない問題を修正しました。

原因：
`feedbackHref()` が参照する `APP_VERSION` が未定義で、設定画面のイベント登録が途中で止まっていました。

修正：
- `APP_VERSION` 定数を追加
- 設定画面のボタン登録を安全化
- フィードバックリンク、開始日変更、今月の準備、リセット、全削除、ホームへ戻るを再確認
